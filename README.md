# ChineseGeography
Support Common Requirements About Chinese Geography
## 接口
- 获取Region的树形数据（省市县镇），可指定树形根节点和叶子节点的级别
- 获取指定Region的直接子节点列表
- 获取指定code的Region
- 获取指定名称的Region，名称可能是标准名称、别名、简称
- 获取capital的Region
- 获取两个Region之间的距离
- 获取指定坐标，方圆N公里内的Region
- 获取指定坐标所在的Region
- 获取非标准输入文本所指的Region
- 获取Region所在的七大自然地理区域
- 获取七大自然地理区域管辖的Region
- 获取邮编所在的Region
- 获取车牌号所在Region
- 获取电话号码区号所在Region
